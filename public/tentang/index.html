Deskripsi Halo Anwar

Halo Anwar adalah website yang dirancang untuk mendokumentasikan hal-hal yang pernah author kerjakan khususnya dibidang Teknologi Informatika. Author akan fokus ke bidang teknologi Machine Learning dan Web Development saja.

Halo Anwar termasuk website statis yang dimana dibangun dengan teknologi HTML, CSS, dan Javascript saja. Dan untuk layanan hosting, author menggunakan layanan gratis dari Google untuk menghosting website statis ini yaitu Firebase Google.

Teknologi Halo Anwar

Berikut beberapa teknologi yang saya butuhkan untuk membuat Halo Anwar :

  1. HTML (HyperText Markup Language)

    Disini author menggunakan teknologi HTML untuk membuat kerangka atau struktur halaman web Halo Anwar. Author kewalahan karena website Halo Anwar masih statis ðŸ˜…, tidak ada sistem templating untuk memudahkan jika author ingin mengubah elemen HTML maka author harus mengubah disemua file HTML.

  2. TailwindCSS

    Untuk memberikan kesan indah pada halaman website Halo Anwar, author menggunakan teknologi TailwindCSS. Kenapa author memilih menggunakan TailwindCSS daripada Bootstrap, karena author bisa bebas berkreasi untuk mengatur halaman web, kalo Bootstrap author merasa terlalu template banget.

  3. Javascript

    Tidak lupa juga untuk menambahkan kesan interaktif pada Halo Anwar, author menggunakan Javascript. Bukan cuma agar terkesan interaktif, author menggunakan Javascript juga untuk melakukan deployment Model Machine Learning yang akan author bahas diteknologi selanjutnya.

  4. TensorFlow JS

    Author menggunakan TensorFlow.js untuk memproses model Machine Learning yang sudah dilatih sebelumnya. Sebelum melakukan deployment dengan TensorFlow.js, author melatih model menggunakan Python dan setelah dirasa akurasi model cukup bagus maka author akan mengconvert model yang sudah dilatih tersebut ke model.json dengan TensorFlow.js.

  5. Firebase Google

    Author menggunakan layanan Firebase dari Google untuk hosting website Halo Anwar. Firebase memiliki banyak layanan seperti Cloud Storage, Machine Learning, Hosting, Realtime Database dan masih banyak lagi. Untuk bisa melakukan hosting author cukup menggunakan layanan hosting saja. Kekurangan dari hosting Firebase, kita tidak bisa melihat code yang kita hosting di Firebase daripada Github Pages yang bisa melihat codenya.
